⬡ GitHub — How to Update the Live Website

https://jriveralegal.github.io/case-file-
1. Before downloading a new index.html — first DELETE the old index.html from your Downloads folder. This prevents Windows from saving it as "index (1).html".
2. Download the new index.html from Claude
3. Go to github.com/jriveralegal/case-file-
4. Click Add file → Upload files
5. Drag and drop index.html onto the upload zone
6. Type commit message: Update [date]
7. Click green Commit changes
8. Wait 2–3 minutes → check the live URL above
⚠️ If site shows blank — it means GitHub Pages is looking for "index.html" but the file was uploaded as "index (4).html". Always check filename before uploading.
